Kinetic Educator is a high-energy, real-time quiz platform designed for seamless classroom interaction. Here are the main features and unique "addons" built into the system:
🚀 Core Features
Real-Time Live Quizzing: Built on Firebase Firestore, allowing students to see questions and teachers to see live "hearts-beating" progress instantly.
Teacher Dashboard: Comprehensive controls to create, edit, and launch quizzes, as well as monitor live room statistics.
Student Participation (No-Login): Students can join instantly via a Room Code or Direct Link using just their Name and Roll Number.
Smart Question Management: Support for Multiple Choice, True/False, Multiple Correct, and Paragraph answers with individual timers for each question.
Automatic Grading & Scoreboard: Instant calculation of scores and time taken, with a leaderboard for the teacher to display.
➕ Unique "Addons" & Technical Polish
Session Locking (Device Handoff): Enables students to resume their quiz if they accidentally close their browser or switch tabs, while preventing two people from using the same roll number simultaneously.
Dynamic Question Shuffling: Each student gets a unique question and option order to prevent peer interference.
Away Detection: Real-time alerts on the teacher's dashboard if a student switches tabs or leaves the browser during the quiz.
Universal Roll Number Support: High-compatibility mode that accepts any roll number format (e.g., 123/CS/2026) used by different institutions.
Bento-Grid Design UI: A polished, modern interface using Inter and Space Grotesk typography for a "tech-forward" educational feel.
